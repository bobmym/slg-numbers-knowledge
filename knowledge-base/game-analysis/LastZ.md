## 🔄 2026-03-19 首次内容填充

### 基础信息

- **开发商：** Florere Game
- **游戏类型：** 融合类 SLG（射击小游戏 + 生存题材 + SLG）
- **题材：** 丧尸生存（海外最热门题材之一）
- **发行时间：** 2025年

### 市场表现

- 2025年4月，随着《Dark War》《Last Z》全球收入增长，带动Florere Game收入增长 **21%**，首次进入Sensor Tower中国手游发行商全球收入榜前十，位列4月榜单
- 2025年7月冲上出海手游榜 **第7名**
- 截至2025年11月升至出海手游榜 **第4位**

### 核心设计逻辑

- **副玩法买量：** 选择海外最热门的丧尸生存题材，通过经典的射击小游戏导量
- **Thronefall-like路径：** 与Kingshot类似，借鉴成熟的副玩法（射击）作为吸量点
- 2025年出海爆款代表之一，验证了"休闲化SLG + 射击小游戏"融合路径的可行性
