> **说明**：本文为两款游戏知识库的横向对比篇，旨在提炼设计共性、差异与可借鉴经验。供数值策划立项参考。

# 二、玩法融合路径对比

## 2.1 X要素的"入口设计"

### 无尽冬日：模拟经营 → SLG

```
入口（模拟经营）
  ├─ 冰雪温度系统（生存压力）
  ├─ 建筑产出（资源正循环）
  └─ 冒险远征（英雄经验来源）
       ↓ 自然过渡
SLG层
  ├─ 联盟活动（GVG核心）
  ├─ 大地图（PVP/PVE）
  └─ 赛季制内容
```

**关键设计**：
- 模拟经营阶段**没有战斗没有战争**，纯粹成长
- 城建是**双轨制**（养成建筑 + SLG建筑），减少卡点
- 英雄是核心战力锚点，城建服务于英雄培养

### Last War：超休闲 → SLG

```
入口（超休闲，5分钟）
  ├─ 数学射击/合并玩法（爽感强）
  └─ 承诺=兑现（广告=游戏）
       ↓ 自然过渡
城建设定（放置成长）
  ├─ 资源收集 + 部队训练
  └─ Frontline（合并战斗）
       ↓ 并行
SLG层
  ├─ 联盟活动（僵尸围城）
  └─ GVG联盟战
```

**关键设计**：
- 超休闲入口是**广告素材本身**，最大化转化率
- Frontline是独立模块，不是城建的附加工具

## 2.2 SLG深度接入点对比

| 对比维度 | 无尽冬日 | Last War |
|----------|----------|----------|
| **首次GVG** | 相对较晚（城建后）| 相对较早（联盟活动频繁）|
| **英雄定位** | 核心战力100% | 战力辅助层 |
| **战力核心杠杆** | 英雄星级（高付费深度）| 装备强化（无限深度）|
| **城建节奏** | 极快（模拟经营密度）| 中等（标准SLG）|
| **Pay to Win 感知** | 中（英雄导向）| 中高（装备导向）|

# 三、数值设计理念对比

## 3.1 资源与成长曲线

### 无尽冬日：充裕资源 + 英雄为王

| 维度 | 设计选择 | 数值效果 |
|------|----------|----------|
| 资源压力 | **极低**（刻意消除卡点）| 玩家不焦虑，城建不痛苦 |
| 成长爽感 | **英雄强化**（频繁小奖励）| 英雄升级=立即变强 |
| 核心杠杆 | **英雄星级**（稀有卡获取）| 付费天花板高 |
| 数值曲线 | 前期极快 → 中期稳 → 后期英雄主导 | 英雄是唯一真正的"质变点" |

### Last War：适度稀缺 + 装备为王

| 维度 | 设计选择 | 数值效果 |
|------|----------|----------|
| 资源压力 | **中等**（标准SLG）| 有一定稀缺性，但不严苛 |
| 成长爽感 | **T10晋升**（兵种最大质变）+ 装备强化 | 双轨爽感 |
| 核心杠杆 | **装备强化**（无限深度）| 几乎没有天花板 |
| 数值曲线 | 装备驱动无限发散 | 高付费用户有近乎无限的空间 |

### 核心洞察：两种付费深度的设计哲学

| | 无尽冬日 | Last War |
|--|---------|----------|
| **付费深度锚点** | 英雄获取（稀有=强）| 装备强化（无稀缺上限）|
| **Free-to-Play友好度** | 较高（资源不紧张）| 中等（资源有一定压力）|
| **鲸鱼天花板** | 英雄升星有上限，后期趋于平稳 | 装备可无限强化，天花板极高 |
| **ARPDAU** | $1.08（较低）| $2.47（高131%）|

## 3.2 战力构成对比

### 无尽冬日战力构成

```
总战力 = 英雄星级 × 等级 × 装备 + 科技 + 联盟科技
```

- **Hero Quality Gap（英雄品质差距）** 是最大变量
- 城建/兵种战力差距相对次要
- 策略空间：英雄协同组合（盾+矛+弓等）

### Last War战力构成

```
总战力 = 装备强化 + HQ等级 + 兵种T度 + 科技 + 联盟科技
```

- **装备强化差距** 是最大变量
- 装备强化有近乎无限的上限（适合重度付费）
- 策略空间：兵种T度规划 + 装备分配

### 数值策划对比总结

| 设计维度 | 无尽冬日 | Last War |
|----------|----------|----------|
| 核心成长驱动 | 英雄（稀有品）| 装备（无限刷）|
| 策略深度来源 | 英雄协同 + 阵型 | 兵种规划 + 装备分配 |
| 免费玩家体验 | 较好（资源充裕）| 中等（资源有一定压力）|
| 付费爆发点 | 英雄召唤活动 | 装备强化礼包 |
| ARPDAU潜力 | 低→中 | 高 |

# 四、活动体系对比

## 4.1 活动节奏对比

| 维度 | 无尽冬日 | Last War |
|------|----------|----------|
| 日活动 | 猎熊行动（每日两次）| 僵尸围城（多次/周）|
| 周活动 | 兵工厂争夺战 / 堡垒战 | 联盟战 |
| 月活动 | SVS（跨服战，月度大事件）| 赛季制 + 联盟战赛季 |
| 付费活动 | 限时召唤up + 礼包 | 限时礼包 + Live Ops密集 |

## 4.2 活动设计共性

两款产品虽然X要素不同，但活动体系的**设计原则高度一致**：

### 共性一：日有锚点

- 每日低门槛活动（猎熊 / 僵尸围城）
- 目的：**建立每日登录习惯**
- 奖励轻量但稳定

### 共性二：周有对抗

- 联盟GVG活动（兵工厂 / 联盟战）
- 目的：**制造社交粘性，制造联盟归属感**
- 有指挥 vs 无指挥产生明显分化

### 共性三：月有大事件

- 版本级GVG（SVS / 赛季联盟战）
- 目的：**制造最高潮的社交体验，形成内容消费的周期性期待**
- 大战前是天然付费窗口

### 共性四：付费节奏配合内容节奏

| 时间节点 | 付费设计 |
|----------|----------|
| 活动前 | 强化道具礼包（临时紧迫感）|
| 活动后 | 奖励结算 + 新内容预告 |
| 月度大事件前 | 高级礼包 + 限时召唤 |
| 节日节点 | 专属皮肤/礼包 |

# 五、广告素材策略对比

## 5.1 获客钩子对比

| 维度 | 无尽冬日 | Last War |
|------|----------|----------|
| 广告核心视觉 | 冰雪末日 + 模拟经营画面 | 数学射击/合并爽感 |
| 目标用户 | 泛用户（女性偏高）| 策略用户 + 男性偏高 |
| 广告一致性 | 模拟经营为主 | 超休闲玩法=实际游戏内容 |
| 转化质量 | 中（休闲用户需转化到SLG）| 较高（广告=前5分钟体验）|

## 5.2 对数值策划的启示

**广告素材反映的是游戏的**"承诺体验"**：
- 无尽冬日承诺的是"冰雪世界的生存与成长"→ 转化后用户预期是成长
- Last War承诺的是"合并爽感"→ 转化后用户已有初步的爽感体验

**两种路径都能成功**，核心在于：
1. 广告承诺必须在游戏内真实存在
2. 转化后的第二层体验必须有足够的留存深度

# 六、SLG+X赛道的设计结论

## 6.1 成功要素提炼

### 要素一：X要素必须是"独立的爽感闭环"

成功的SLG+X产品，X要素本身就是一个**完整的爽感系统**，不依赖SLG层而存在：
- 无尽冬日：模拟经营=建造+资源=成长爽感 ✅
- Last War：合并=数字倍增=即时爽感 ✅

> 如果X要素只是SLG的"新手引导"，而不是独立的价值载体，则很难实现破圈。

### 要素二：X→SLG的过渡必须自然无断裂

| 产品 | 过渡设计 |
|------|----------|
| 无尽冬日 | 城建建筑逐渐解锁SLG内容，体验渐变 ✅ |
| Last War | Frontline是独立模块，与城建并行，体验分层 ✅ |

### 要素三：核心付费深度必须有明确的"载体"

| 产品 | 付费深度载体 | 特点 |
|------|------------|------|
| 无尽冬日 | 英雄星级 | 稀缺=强，可量化，天花板明确 |
| Last War | 装备强化 | 无上限，适合重度付费 |

### 要素四：留存必须靠"日活动锚点"

两款产品都把**日活动**作为留存核心：
- 不是靠内容消费量（传统MMO思路）
- 而是靠**每日固定社交事件**形成习惯

## 6.2 对数值策划的核心借鉴

### 数值框架设计

```
1. 确定核心爽感来源（X要素是什么）
2. 确定核心付费竖井（什么数值差距决定付费）
3. 搭建"日锚点+周对抗+月大事件"活动节奏
4. 资源稀缺性要服务于爽感节奏，而非本末倒置
```

### 付费深度设计

| 问题 | 借鉴 |
|------|------|
| 如何设计付费天花板？ | Last War：无限装备强化；无尽冬日：英雄星级上限 |
| 如何平衡免费玩家？ | 无尽冬日：资源充裕，护付费体验 |
| 付费爆发点在哪？ | 大型GVG活动前（新英雄/装备礼包）|

### 英雄/战力迭代策略

- **无尽冬日模式**：靠英雄迭代（新英雄=新强度天花板）
- **Last War模式**：靠装备迭代（新装备等级=新强度天花板）

两种方式都可以，关键是与**活动节奏**配合。

# 七、风险与局限性

| 维度 | 无尽冬日风险 | Last War风险 |
|------|------------|-------------|
| **留存天花板** | D30 8%尚可，但依赖版本更新 | D30 4%，留存危机需要解决 |
| **创意老化** | 冰雪题材长期吸引力待观察 | Frontline合并玩法可能快速被复制 |
| **市场竞争** | SLG+X赛道已涌入大量跟随者 | 中国市场未进入（机会=风险）|
| **平台依赖** | 微信小游戏生态 | 纯App生态 |
| **监管风险** | 国内版合规要求 | 无中国版，少一个收入来源 |
