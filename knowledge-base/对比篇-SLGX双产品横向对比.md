> **说明**：本文为两款游戏知识库的横向对比篇，旨在提炼设计共性、差异与可借鉴经验。供数值策划立项参考。

# 一、核心定位对比

## 1.1 产品定位光谱

| 维度 | 无尽冬日 / Whiteout | Last War: Survival |
|------|---------------------|-------------------|
| **融合玩法（X）** | 冰雪模拟经营 | 数字门跑酷射击 |
| **SLG深度** | 4X策略（强）| 4X策略（强）|
| **对外形象** | 冰雪末日生存游戏 | 僵尸末日生存游戏 |
| **核心爽感** | 英雄养成 + 温度压力 | 数字倍增 + 放置爆发 |
| **用户形象** | 偏泛用户，女性占比高 | 偏男性策略用户 |
| **市场** | 海外 + 国内（微信小游戏）| 海外为主，无中国版 |
| **团队基因** | 点点互动（发行代理转自研）| First Fun（4X老将创业）|

## 1.2 ⭐ SLG+X 路径分歧：两种截然不同的融合哲学

根据知乎文章深度分析，两款产品虽然都标榜"SLG+X"，但**X要素的定位存在本质差异**：

### 无尽冬日：SLG为主，X是入门体验

```
X要素（模拟经营）
    ↓ 过渡
SLG层（核心价值）
```

- 模拟经营是**新手引导和门槛降低**
- 核心价值仍是SLG的GVG对抗
- X玩法服务于SLG留存
- **核心钩子**：生存叙事的感性胜利，激发玩家"保护家园的责任感"

### Last War：X是核心，SLG是载体

```
X要素（数字门跑酷）
    ↓ 核心吸引
SLG层（变现载体）
```

- 数字门爽感是**真正的差异化卖点**
- 广告素材=游戏前5分钟体验
- SLG负责承接付费深度
- **核心钩子**：即时决策的理性爽感，击中"倍增收益"的直观贪婪感
- **获客优势**：CPI比同类产品低约30%-40%，是SLG赛道的"吸量怪物"

### 路径选择启示

| 问题 | 无尽冬日选择 | Last War选择 |
|------|------------|-------------|
| X要素定位 | 入门体验，降低门槛 | 核心爽感，第一吸引力 |
| 转化用户类型 | 泛用户→SLG用户 | 超休闲用户→SLG用户 |
| 核心收入来源 | 英雄升星（稀缺有上限）| 装备强化（无限深度）|
| 转化节奏 | 浸润式（软着陆）| 爆发式（硬冲击）|
| 适合场景 | 需破圈、吸引泛用户 | 已有核心玩法，寻求差异化 |

# 二、副玩法爽感工程指标

## 2.1 ⭐ 广告承接期：决定生死的"前30分钟"

**核心原则**：
- 投入80%精力聚焦广告承接期
- 用副玩法的爽感（糖衣）包裹SLG的枯燥数值（药丸）
- 通过题材逻辑和建造引导，让玩家不知不觉转化为长线SLG付费用户

## 2.2 爽感的三大工程指标

| 指标 | 具体设计 | 示例 |
|------|----------|------|
| **A. 视觉密度"割草感"** | 数量级的瞬间膨胀、高密度伤害数字、满屏弹幕、成片敌人倒下 | Last War满屏僵尸倒下 |
| **B. 极低"输入输出比"** | 极低成本操作（如滑1下）获得极高反馈（如消灭100个丧尸） | 数字门x2选择 |
| **C. "险胜"节奏控制** | 焦虑（怪物压境）→ 操作（升级/捡道具）→ 释放（瞬间清屏） | Boss战最后一击 |

**公式**：副玩法的爽感 = 视觉上的"暴发户"体验 + 操作上的"傻瓜式"门槛

## 2.3 "我是谁"的代入感构建

| 产品 | 身份赋予 | 转化逻辑 |
|------|----------|----------|
| **无尽冬日** | "指挥官" + 幸存者保护者 | 因为要保护幸存者→必须建基地→训练军队 |
| **Last War** | "指挥官" + 末世幸存者 | 因为是指挥官→不仅要打僵尸→更要建基地、保护幸存者 |

> **风险提示**：欧美RPG玩家非常排斥玩法割裂感

## 2.4 英雄系统的深度绑定

| 对比维度 | 无尽冬日 | Last War |
|----------|----------|----------|
| **传统SLG（COK模式）** | 兵强则强（英雄是挂件）| 兵强则强 |
| **休闲模式** | 人强则强（英雄是本体）| 英雄=跑酷战力核心 |
| **英雄定位** | 核心战力100%（探险/远征双模式）| 战力辅助层，但绑定跑酷体验 |

### 英雄系统的三大统一（Last War）

| 统一 | 说明 |
|------|------|
| **认知统一** | 副玩法控制的英雄 = SLG大地图使用的英雄 |
| **验证统一** | 副玩法是检验英雄强度的"考场" |
| **唯一数据源** | 同一套数据库，同一套养成系统 |

# 三、玩法融合路径对比

## 3.1 X要素的"入口设计"

### 无尽冬日：模拟经营 → SLG

```
入口（模拟经营）
  ├─ 冰雪温度系统（生存压力）
  ├─ 建筑产出（资源正循环）
  └─ 冒险远征（英雄经验来源）
       ↓ 自然过渡
SLG层
  ├─ 联盟活动（GVG核心）
  ├─ 大地图（PVP/PVE）
  └─ 赛季制内容
```

**关键设计**：
- 模拟经营阶段**没有战斗没有战争**，纯粹成长
- 城建是**双轨制**（养成建筑 + SLG建筑），减少卡点
- 英雄是核心战力锚点，城建服务于英雄培养

### Last War：超休闲 → SLG

```
入口（超休闲，5分钟）
  ├─ 数学射击/数字门玩法（爽感强）
  └─ 承诺=兑现（广告=游戏）
       ↓ 自然过渡
城建设定（放置成长）
  ├─ 资源收集 + 部队训练
  └─ Frontline（数字门战斗）
       ↓ 并行
SLG层
  ├─ 联盟活动（僵尸围城）
  └─ GVG联盟战
```

**关键设计**：
- 超休闲入口是**广告素材本身**，最大化转化率
- Frontline是独立模块，不是城建的附加工具

## 3.2 ⭐ 转化心流对比：软着陆 vs 硬冲击

### 无尽冬日：软着陆（浸润式转化）

| 阶段 | 机制 | 玩家心理 |
|------|------|----------|
| Chapter 1-3 | 几乎全是生存模拟，不谈战争，只谈温暖 | 建立"家园归属感" |
| Chapter 6 | 分水岭：人口增加，粮食/木材出现巨大"人为缺口" | 玩家点击"升级熔炉"却发现资源不足，等待时间变2小时 |
| **转化瞬间** | 挫败感 → 首充黄金窗口 | 玩家被迫付费加速 |

### Last War：硬冲击（爆发式转化）

| 阶段 | 机制 | 玩家心理 |
|------|------|----------|
| 前期 | 技巧可以弥补战力 | "我很强"、"游戏很简单有趣" |
| 中后期 | 技巧无法弥补战力鸿沟 | 满屏僵尸冲破防线 |
| **转化瞬间** | 系统弹出"SSR英雄金"礼包 | 付费冲动释放更早 |

## 3.3 ⭐ 体验阶段模型对比

### 无尽冬日：单阶段平滑过渡

| 阶段 | 时长 | 特点 |
|------|------|------|
| 模拟经营期 | 前3-7天 | 放置成长，无战争 |
| SLG导入期 | 第5天开始 | 一代盾+最强领主，GVG启蒙 |
| 深度SLG | 第14天后 | 完整联盟生态 |

### Last War：三阶段漏斗模型

| 阶段 | 时长 | 爽感类型 |
|------|------|----------|
| 跑酷射击 | 0-5分钟 | 即时爽感，广告级体验 |
| 塔防Boss战 | 1小时内 | 数字倍增，多巴胺刺激 |
| 长线SLG | 长期 | 联盟对抗，装备追求 |

> Last War的三阶段设计形成**"承诺→兑现→深化"**的漏斗，确保用户不会在第一分钟流失。

## 3.4 核心战力矩阵对标

| 维度 | 无尽冬日 (WS) | Last War (LW) |
| --- | --- | --- |
| **生存压力** | 极寒/罢工/疾病 | 尸潮/逻辑失误/火力不足 |
| **数值锚点** | **熔炉等级** (Furnace) | **总部等级** (HQ) |
| **变现卡点** | 第6章：煤矿缺口与人口压力 | HQ 12级：英雄战力无法跨越数字门 |
| **社交粘性** | 极高（共同防御寒冬） | 高（大规模领地战） |

# 四、战斗系统深度对比

## 4.1 无尽冬日：双模式融合设计

根据知乎文章拆解，无尽冬日采用**两种战斗模式融合**：

### 探险战斗（即时制）
- **形式**：英雄带少量小兵上阵
- **特点**：英雄可直接造成伤害
- **类比**：类似AFK类游戏的英雄带小兵战斗模式
- **设计目的**：增加前期爽感，提供即时战斗反馈

### 远征战斗（回合制）
- **形式**：在大世界发生的战斗
- **特点**：英雄不会下场战斗造成具体伤害
- **设计**：通过触发技能，给小兵提供buff加成
- **设计目的**：强调士兵重要性，保持SLG策略深度

### 双模式设计优势

| 优势 | 说明 |
|------|------|
| 增加英雄多样性玩法 | 针对不同战斗形式设计英雄，增加英雄的就业需求 |
| 强调士兵重要性 | 大世界战斗基础属性来源于士兵，士兵属性来源于等级 |
| 隔离设计 | 确保后期英雄养成不会影响士兵需求，两者独立发展 |

## 4.2 Last War：Frontline合并战斗

- Frontline是**独立模块**，与城建并行
- 合并=数字倍增=即时爽感
- 英雄系统作为战力辅助层（坦克/导弹车/飞机三大类）
- 三大类存在克制链：坦克→克制导弹→克制飞机→克制坦克

# 七、活动体系对比

## 7.1 ⭐ 无尽冬日：11种核心活动生态

根据知乎文章分析，无尽冬日设计了**完整的活动生态**：

| 活动类型 | 代表活动 | 周期 | 设计原则 |
|----------|----------|------|----------|
| PVE集结 | 猎熊行动 | 每日×2 | **先集体PVE建立情感** |
| 免费获取 | 吉娜的反击 | 每日 | 刷升星石，免费5星射手 |
| 公会守塔 | 疯狂的乔伊 | 每日 | 比拼防守 |
| GVG占点 | 兵工厂争夺战 | 每周 | 类似魔兽阿拉希盆地 |
| GVG据点 | 堡垒争夺战 | 周末 | 抢夺大地图资源点 |
| 本服比拼 | 最强领主 | 赛季 | 资源消耗大乱斗 |
| 跨服策略 | 联盟争霸赛 | 赛季 | 田忌赛马 |
| 王城争夺 | 决战王城 | 赛季 | 联盟终极荣誉 |
| 超大GVG | SVS跨服战 | 每月 | 版本级事件 |
| 休闲调节 | 钓鱼 | 每日 | 缓解SLG紧张氛围 |
| 养成奖励 | 冒险远征 | 长期 | 英雄经验主要来源 |

### 核心设计原则：先PVE后GVG

> **"前7天避免天天干仗，通过PVE建立感情链接。真正氪金活动「最强领主」在第5天开始，配合一代盾投放，大幅提高LTV。"**

遵循**「先集体PVE建立情感→形成团队纽带→扩大竞争」**的节奏。

## 7.2 Last War：赛季驱动模式

| 赛季 | 时间 | 效果 |
|------|------|------|
| 第一赛季 | 2024年5月 | 产品启动 |
| 第二赛季 | 2024年9月 | 巨大收入增长 |
| 此后 | 每隔数月 | 稳定开启新赛季，形成可预测收入波峰 |

### 赛季三要素
1. **限时活动**（新主题/玩法模式）
2. **竞争**（联盟跨服对战或区服竞赛）
3. **消耗**（限时货币，大幅折扣礼包）

## 7.3 活动设计共性

两款产品虽然X要素不同，但活动体系的**设计原则高度一致**：

### 共性一：日有锚点

- 每日低门槛活动（猎熊 / 僵尸围城）
- 目的：**建立每日登录习惯**
- 奖励轻量但稳定

### 共性二：周有对抗

- 联盟GVG活动（兵工厂 / 联盟战）
- 目的：**制造社交粘性，制造联盟归属感**
- 有指挥 vs 无指挥产生明显分化

### 共性三：月有大事件

- 版本级GVG（SVS / 赛季联盟战）
- 目的：**制造最高潮的社交体验，形成内容消费的周期性期待**
- 大战前是天然付费窗口

### 共性四：付费节奏配合内容节奏

| 时间节点 | 付费设计 |
|----------|----------|
| 活动前 | 强化道具礼包（临时紧迫感）|
| 活动后 | 奖励结算 + 新内容预告 |
| 月度大事件前 | 高级礼包 + 限时召唤 |
| 节日节点 | 专属皮肤/礼包 |

# 八、生态干预与运营

## 8.1 ⭐ 联盟管理"七三法则"

| 法则 | 说明 |

# 九、广告素材策略对比

## 9.1 获客钩子对比

| 维度 | 无尽冬日 | Last War |
|------|----------|----------|
| 广告核心视觉 | 冰雪末日 + 模拟经营画面 | 数学射击/数字门爽感 |
| 目标用户 | 泛用户（女性偏高）| 策略用户 + 男性偏高 |
| 广告一致性 | 模拟经营为主 | 超休闲玩法=实际游戏内容 |
| 转化质量 | 中（休闲用户需转化到SLG）| 较高（广告=前5分钟体验）|
| **CPI优势** | — | 比同类产品低约30%-40% |

## 9.2 Last War广告方法论

根据知乎文章分析，Last War形成了**工业流水线闭环**：

```
买量验证的副玩法母题
    ↓
广告与新手体验一致
    ↓
渠道窗口期集中投放
    ↓
素材小迭代延长寿命
```

- 核心买量素材中数字射击小游戏占展示量 **50%以上**
- 承诺=兑现，避免用户"被骗"的感觉
- 素材迭代：换场景色调、调整敌人外观、加入明星物料

## 9.3 对数值策划的启示

**广告素材反映的是游戏的**"承诺体验"**：

- 无尽冬日承诺的是"冰雪世界的生存与成长"→ 转化后用户预期是成长
- Last War承诺的是"数字倍增爽感"→ 转化后用户已有初步的爽感体验

**两种路径都能成功**，核心在于：
1. 广告承诺必须在游戏内真实存在
2. 转化后的第二层体验必须有足够的留存深度

# 十、战略层面抉择

## 10.1 大R模型 vs 中R模型

| 特征 | Last War（大R模型） | 无尽冬日（中R模型） |

| 资源分配 | 说明 |
|----------|------|

4. **对人性的极致操控**：大R虚荣心、中R归属感、小R生存欲精确转化为付费点和留存率

# 十一、SLG+X赛道的设计结论

## 11.1 成功要素提炼

### 要素一：X要素必须是"独立的爽感闭环"

成功的SLG+X产品，X要素本身就是一个**完整的爽感系统**，不依赖SLG层而存在：
- 无尽冬日：模拟经营=建造+资源=成长爽感 ✅
- Last War：数字门=倍增=即时爽感 ✅

> 如果X要素只是SLG的"新手引导"，而不是独立的价值载体，则很难实现破圈。

### 要素二：X→SLG的过渡必须自然无断裂

| 产品 | 过渡设计 |
|------|----------|
| 无尽冬日 | 城建建筑逐渐解锁SLG内容，体验渐变 ✅ |
| Last War | Frontline是独立模块，与城建并行，体验分层 ✅ |

### 要素三：核心付费深度必须有明确的"载体"

| 产品 | 付费深度载体 | 特点 |
|------|------------|------|
| 无尽冬日 | 英雄星级 | 稀缺=强，可量化，天花板明确（5代上限）|
| Last War | 装备强化+英雄 | 无上限，适合重度付费 |

### 要素四：留存必须靠"日活动锚点"

两款产品都把**日活动**作为留存核心：
- 不是靠内容消费量（传统MMO思路）
- 而是靠**每日固定社交事件**形成习惯

### 要素五：社交节奏先PVE后GVG

| 产品 | 社交节奏设计 |
|------|-------------|
| 无尽冬日 | 前7天避免GVG，先PVE建立情感 ✅ |
| Last War | 联盟活动密集，但转化用户已有预期 ✅ |

## 11.2 路径选择决策树

```
立项决策
    │
    ├─ 是否有差异化超休闲玩法？
    │       │
    │       ├─ 是 → 参考Last War模式（X为核心）
    │       │         └─ 广告即内容，最大化转化
    │       │
    │       └─ 否 → 参考Whiteout模式（SLG为主）
    │                 └─ X作为入门体验，降低门槛
    │
    └─ 目标用户是谁？
            │
            ├─ 泛用户 → Whiteout（女性占比高）
            │
            └─ 核心玩家 → Last War（ARPDAU高）
```

## 11.3 对数值策划的核心借鉴

### 数值框架设计

```
1. 确定核心爽感来源（X要素是什么）
2. 确定核心付费竖井（什么数值差距决定付费）
3. 搭建"日锚点+周对抗+月大事件"活动节奏
4. 资源稀缺性要服务于爽感节奏，而非本末倒置
5. 设计社交节奏：先PVE建立情感，后GVG扩大竞争
```

### 付费深度设计

| 问题 | 借鉴 |
|------|------|
| 如何设计付费天花板？ | Last War：无限装备强化；无尽冬日：英雄5代上限 |
| 如何平衡免费玩家？ | 无尽冬日：资源充裕，护付费体验 |
| 付费爆发点在哪？ | 大型GVG活动前（新英雄/装备礼包）|
| 如何延长生命周期？ | 无尽冬日：代差英雄体系；Last War：赛季制 |

### 英雄/战力迭代策略

| 策略 | 无尽冬日模式 | Last War模式 |
|------|-------------|-------------|
| 迭代方式 | 英雄代差（新英雄=新天花板）| 装备等级上限（新等级=新天花板）|
| 生命周期 | 5代英雄分割内容 | 赛季推动新装备需求 |
| 毕业设计 | 英雄满星毕业，可追新 | 装备无毕业，永续追求 |
