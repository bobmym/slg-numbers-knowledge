# 一、现状诊断

## 1.1 问题定义

《疯狂水世界》作为休闲SLG 2.0的代表产品，核心挑战在于：如何让模拟经营（买量钩子）、卡牌养成（付费驱动）、休闲SLG（社交粘性）三大模块有机衔接，同时平衡大R与非大R的体验差异，避免两端玩家同步流失。

## 1.3 关键发现

**发现1：产品定位为"模拟经营的买量钩子，SLG的皮，卡牌养成的骨"** 🟢 高可信度，来源为文章原文定义

- 卡牌养成+团队构筑在玩家体验动线上占据绝对主导位置
- 模拟经营承担用户导入与留存锚点功能
- SLG仅作为联盟社交模块，不承担核心数值供给

**发现2：城建资源与战略资源完全解耦是核心设计创新** 🟢 高可信度，来源为文章原文

- 城建资源（7类材料）只能用于城内工厂生产，与战略资源无关
- 战略资源（科技点/种子/燃油）只与城战胜负挂钩
- 玩家放弃SLG不会导致"弹尽粮绝"，仅影响城内加工产线

**发现3：放置挂机模块是留存"定海神针"** 🟢 高可信度，来源为文章原文

- D0-D60生命周期中，玩家注意力集中于配队、调站位、闯关
- 挂机产出等效月价值2000元+，大幅稀释SLG挫败感
- 非付费玩家可通过挂机追赶大部分养成进度

**发现4：红卡滚卡机制驱动大R持续付费** 🟢 高可信度，来源为文章原文

- 每赛季推出新红卡角色，提供新机制而非单纯更高数值
- 大R需2-4周通过限购礼包拉满星级和专武
- 非大R（VIP10/3000元以下）游戏体验基本趋同

**发现5：城战时间压缩有效压制负反馈** 🟢 高可信度，来源为文章原文

- 仅每晚20:00-21:00开放城战，其余时间SLG功能不开放
- 所有玩家舰船耐久相同，大R仅多5次交战机会
- PVP奖励被普遍削减，低于日常数值投放一个数量级

# 二、竞品对比

> ⚠️ 分类勘误：本节对"X + SLG"与"SLG + X"框架的原始定义有误，已按李佳指正修正。核心区分维度为：①玩家城池是否体现在SLG玩法中；②基础资源（城建资源）的获取途径和用途。

修正后的定义：

- **X + SLG** = 疯狂水世界模式。玩家城池**不体现**在SLG玩法中；城建资源（7类材料）用于城内工厂生产，**与战略资源不完全绑定**。
- **SLG + X** = 无尽冬日、奔奔王国模式。SLG为**主体**，城建/塔防/模拟经营作为**前置钩子**压低CPI。玩家有独立城池，城建资源与战略资源深度绑定。

## 2.3 主要差距分析

**差距1：模拟经营深度远超同类产品**

- 7类细分资源 + 网状生产链 + 动态平衡机制
- 对比《指尖无双》模拟经营较浅，《咸鱼之王》无模拟经营
- 初期完全不呈现SLG战斗感，用纯模拟经营承接玩家
- 差异化优势：题材新颖（海洋废土+废品孤岛）

**差距2：城建/战略资源解耦程度**

- 传统SLG（率土、COK框架）：城内城外共享资源，城破=弹尽粮绝
- 疯狂水世界：放弃SLG仅影响联盟领地，不影响个人城内产线
- 《无尽冬日》《奔奔王国》仍存在城池概念，资源未完全解耦

**差距3：卡牌付费深度 vs SLG付费深度**

- 疯狂水世界：卡牌为核心付费驱动，SLG弱化付费差异
- 无尽冬日：SLG为核心付费驱动，城建为钩子
- 本质差异：付费杠杆集中点不同

# 三、深度调研（四层递进）

## 3.1 模拟经营模块

### L1 框架层：系统定位与结构分类

分类维度 / 设计

循环类型 — 「采集——建造——加工」三级循环

资源类型 — 7类细分材料 + 加工品体系

产出导向 — 建筑升级 + 卡牌养成材料

特色定位 — 买量钩子，初期完全不含SLG气息

### L2 机制层：规则逻辑与交互流程

**采集阶段**

- 主角角色+废品孤岛NPC集体出动采集
- "救人越多，采集越高效"——营救NPC作为升级条件
- 手动钓鱼、拾荒等解压交互

**建造阶段**

- 基地由玩家自行建造
- 置于模拟经营资源循环之后，非前置

**加工阶段**

- 7类材料细分
- 高级加工品由低级加工品生产，需其他品类加工品作为原料
- 加工品等级越高，所需时间越长，越需要英雄养成度提速

**资源动态平衡**

- "供过于求→供不应求"动态平衡
- 基础供应充分、需求严重短缺，将玩家时间引导至其他玩法

### L3 数值层：关键参数

参数 / 数值/设计

资源类型数 — 7类细分材料

加工链层级 — 初级加工→中级加工→高级加工

时间成本 — 加工品等级越高，时间越长

增产机制 — 英雄养成度提速，所有增产集中于1个建筑

### L4 生态层：用户行为与商业节奏

- 模拟经营产出是卡牌养成的资源基底
- 7类资源形成复杂网状结构，摸索成本高
- 非付费玩家可通过模拟经营积累资源，为挂机模块提供基础

---

## 3.2 卡牌养成模块

### L1 框架层：系统定位与结构分类

分类维度 / 设计

卡牌类型 — 日常可获取 / 付费可获取 / 深度付费

养成维度 — 等级、星级、装备、徽章（固定数值）、藏品

核心定位 — 战斗数值来源 + 付费驱动核心

特色设计 — 图鉴收集度反哺模拟经营速度

### L2 机制层：规则逻辑与交互流程

**英雄获取机制**

- 三池分离：日常可获取 / 付费可获取（极低概率免费）/ 深度付费
- 图鉴内橙色角色需招募到一定基数才可解锁，最深达1600次
- 10连抽包含英雄碎片等养成物
- 付费英雄不会在免费池出现

**红卡（超凡）系统**

- 每赛季对应活动转盘获取
- 与混合卡池做隔离处理
- 大R需2-4周通过限购礼包拉满星级和专武
- 提供下级卡牌完全无法比拟的实战机制（非单纯更高数值）

**卡牌与模拟经营联动**

- 图鉴收集度越高 → 模拟经营增产效率越高
- 形成"抽卡→收集→增产→更多资源→继续抽卡"正向飞轮

**挂机模块**

- 核心放置产出系统
- 离线玩家可积累大量挂机收益
- 挂机产出现值为月2000元+，大幅稀释SLG失败挫败感

### L3 数值层：关键参数

参数 / 数值/设计

红卡养成周期（大R）— 2-4周拉满（限购礼包）

放置挂机周产出 — 超648直充等效价值

放置挂机月产出 — 2000元+等效

赛季周期 — 约18天

### L4 生态层：用户行为与商业节奏

- 非大R在卡牌获取上体验趋同，差异主要体现在养成速度
- 红卡提供"新机制"而非"更高数值"，是策略新鲜感来源
- 赛季制确保每18天有新目标，防止长草

## 3.3 休闲SLG模块

### L1 框架层：系统定位与结构分类

分类维度 / 设计

核心定位 — 联盟社交模块，不承担核心数值供给

城战时间 — 每晚20:00-21:00，其余时间SLG功能不开放

舰船耐久 — 所有玩家相同，大R仅多5次交战机会（交换比1:20）

### L2 机制层：规则逻辑与交互流程

**城战设计**

- 仅晚间1小时开放城战，时间极度压缩
- 所有玩家舰船耐久相同（交换比极限1:20）
- 大R仅比普通玩家多5次交战机会
- 联盟为单位参与，非个人城池战

**PVP奖励**

- 奖励被大幅削减，低于日常数值投放一个数量级
- 城战目的：社交荣誉 > 数值收益

**战略资源获取**

- 科技点/种子/燃油：仅与城战胜负挂钩
- 与个人城内产线完全解耦
- 放弃SLG = 不影响城内加工，不弹尽粮绝

### L3 数值层：关键参数

参数 / 数值/设计

城战时长 — 晚间1小时（20:00-21:00）

舰船耐久交换比 — 1:20（极限值）

大R交战次数加成 — 仅多5次

PVP奖励幅度 — 低于日常投放一个数量级

### L4 生态层：用户行为与商业节奏

- SLG是社交锚点，不是数值压力来源
- 联盟为单位，降低个人PVP压力
- 时间压缩让SLG不抢占模拟经营/卡牌养成的时间

# 四、商业化设计亮点

## 4.1 付费分层结构

**免费玩家** — 通过模拟经营+挂机维持进度，付费差距主要体现在养成速度

**中R（VIP10/3000元以下）** — 体验与免费玩家基本趋同

**大R（万元以上）** — 红卡满星满专武，2-4周可拉满

## 4.2 红卡滚卡机制

- 每赛季推出一张新红卡，提供新机制
- 大R通过限购礼包在2-4周内满星满专武
- 新红卡替代旧红卡 → 上一代红卡逐渐流入中低档卡池
- 形成"大R追新→旧卡贬值→中低R受益"的生态循环

## 4.3 三池分离设计

| 卡池类型 | 获取方式 | 定位 |
|----------|----------|------|
| 日常池 | 免费/钻石可获取 | 基础体验保障 |
| 付费池 | 付费获取，极低概率免费 | 中R付费点 |
| 深度付费池 | 活动转盘/专属礼包 | 大R专属，红卡隔离获取 |

## 4.4 商业化节奏

- 赛季周期：约18天
- 每赛季：新红卡上线 → 大R集中付费 → 新红卡进入混合池 → 中低R可获取旧红卡
- 城战+PVP：弱化付费差距感知，不作为主要收入来源

# 五、设计结论与借鉴

## 5.1 核心借鉴（数值策划角度）

**资源解耦是核心创新**：城建资源与战略资源解耦，让SLG失败不至于"弹尽粮绝"，大幅降低流失率

**付费杠杆集中于卡牌养成**：SLG不承担数值供给，付费差异不在PVP中直接体现，保护非付费玩家体验

**放置挂机作为"追赶机制"**：月等效2000元+的挂机产出，让非付费玩家可追赶大部分进度

**红卡新机制替代新数值**：每赛季新红卡提供新机制而非单纯更高数值，确保策略新鲜感

## 5.2 与无尽冬日的本质差异

| 维度 | 疯狂水世界 | 无尽冬日 |
|------|-----------|----------|
| 品类归属 | X + SLG | SLG + X |
| 核心付费驱动 | 卡牌养成 | SLG城建/英雄 |
| 城池是否进入SLG | 否 | 是 |
| 城建/战略资源耦合 | 完全解耦 | 部分耦合 |
| 非大R留存设计 | 强（挂机追赶）| 强（内容深度）|
